# 収益化セットアップガイド

お掃除ナビの収益化（Google AdSense・楽天アフィリエイト）の設定方法です。

---

## 1. Google AdSense

### 1-1. アカウント申請

1. [Google AdSense](https://www.google.com/adsense/) にアクセス
2. Googleアカウントでログイン
3. 「今すぐ始める」から申請
4. サイトURL（例：`https://cleaning-site-lac.vercel.app`）を登録
5. 審査通過まで数日〜数週間かかる場合があります

### 1-2. 広告ユニットの作成

審査通過後：

1. AdSense管理画面 → **広告** → **広告ユニット**
2. 以下のサイズで広告ユニットを作成：
   - **レクタングル（中）**: 336×280
   - **レクタングル**: 300×250（オプション）
   - **バナー**: 728×90（オプション）

3. 各広告の**スロットID**（`1234567890` のような数字）をメモ

### 1-3. 環境変数の設定

**Vercelの場合：**

1. [Vercel](https://vercel.com) → プロジェクト → **Settings** → **Environment Variables**
2. 以下を追加：

| 変数名 | 値 | 説明 |
|--------|-----|------|
| `PUBLIC_ADSENSE_ENABLED` | `true` | 広告を有効化 |
| `PUBLIC_ADSENSE_CLIENT` | `ca-pub-xxxxxxxx` | 発行者ID（AdSense管理画面で確認） |
| `PUBLIC_ADSENSE_SLOT_MEDIUM` | `1234567890` | 336×280のスロットID |
| `PUBLIC_ADSENSE_SLOT_SMALL` | `1234567890` | 300×250のスロットID（任意） |
| `PUBLIC_ADSENSE_SLOT_BANNER` | `1234567890` | 728×90のスロットID（任意） |

3. **Redeploy** で再デプロイ

**ローカル開発の場合：**

`.env` ファイルを作成（`.env.example` をコピーして編集）

## 2. 楽天アフィリエイト

### 2-1. アフィリエイト登録

1. [楽天アフィリエイト](https://affiliate.rakuten.co.jp/) にアクセス
2. 会員登録・ログイン
3. サイトを登録（審査あり）

### 2-2. 商品リンクの取得

1. 楽天アフィリエイト管理画面 → **ツール** → **リンク作成**
2. 楽天市場で商品を検索
3. 「アフィリエイトリンクを作成」でリンクを取得
4. 取得したURLを記事の `recommendedProducts` に設定

### 2-3. 記事への商品設定

記事のMarkdownフロントマターに追加：

```yaml
recommendedProducts:
  - name: 重曹 1kg
    url: https://a.r10.to/xxxxx  # 楽天アフィリエイトのリンク
    image: https://thumbnail.image.rakuten.co.jp/...
    price: "¥398"
  - name: クエン酸 1kg
    url: https://a.r10.to/yyyyy
    image: https://thumbnail.image.rakuten.co.jp/...
    price: "¥598"
```

- **name**: 商品名
- **url**: 楽天アフィリエイト経由の商品URL（必須でリンク化）
- **image**: 商品画像URL（任意）
- **price**: 価格表示（任意）

### 2-4. 規約について

楽天アフィリエイトの規約に従い、以下の表記を自動で表示しています：

> 当サイトは楽天アフィリエイトを利用しています。リンクから購入すると、当サイトに紹介料が入る場合があります。

## 4. 確認方法

1. 環境変数設定後、Vercelで再デプロイ
2. 本番サイトで記事ページを開く
3. AdSense: 広告が表示されることを確認
4. アフィリエイト: 商品リンクが正しく表示されることを確認

---

## 5. トラブルシューティング

### 広告が表示されない

- `PUBLIC_ADSENSE_ENABLED` が `true` か確認
- 発行者ID・スロットIDが正しいか確認
- AdSenseの審査が通過しているか確認
- 広告ブロッカーを無効にして確認

### アフィリエイトリンクが効かない

- URLが楽天アフィリエイト経由になっているか確認
- リンクに `rel="sponsored"` が付与されている（自動設定済み）

## 6. A8.net 登録用 サイト説明文

A8.net にサイトを追加する際に、そのままコピーして使える説明文です。必要に応じてサイトURLや文字数に合わせて編集してください。

### サイト概要（短め・200字前後）

```
「お掃除ナビ」は、キッチン・浴室・リビングなど場所別、油汚れ・水垢・カビなど汚れ別、季節別にお掃除の方法や掃除のコツをまとめた情報サイトです。換気扇・排水口・窓の掃除など具体的な手順を記事で解説しており、重曹・クエン酸・酸素系漂白剤の使い分けなど共通知識も掲載しています。家事・掃除に役立つノウハウを提供し、読者のお掃除の参考になるコンテンツを運営しています。
```

### サイト概要（やや詳しめ・400字前後）

```
「お掃除ナビ」は、お掃除の方法や掃除のコツを、場所・汚れ・季節の3軸で探せる情報サイトです。

【コンテンツ】
・場所で探す：キッチン、浴室、リビング、玄関など場所別の掃除方法
・汚れで探す：油汚れ、水垢、カビ、シミなど汚れ別の落とし方
・季節で探す：春・夏・年末など季節に合わせたお掃除や大掃除のコツ
・共通知識：重曹・クエン酸・酸素系漂白剤の使い分けなど、掃除の基本知識
・ブログ：平日10分でできるミニ掃除ルーティンなど実践的な記事

各記事では具体的な手順や使えるグッズを紹介しており、読者が「やってみよう」と思える内容を心がけています。すでに楽天アフィリエイトで記事に合った商品を紹介しており、A8.net の広告・ASP案件とも相性の良いテーマで、記事内容に沿った商品・サービスを自然な形で掲載する方針です。
```

### アフィリエイトの利用方法（記載用・短め）

```
記事内の「この作業で使えるおすすめグッズ」ブロックや、共通知識・まとめ記事において、記事テーマに合った洗剤・掃除用具・家電などの商品リンクを掲載します。読者の役に立つ情報と関連性の高い商品紹介に努め、規約に従って表記・リンク設定を行います。
```
